Architecture Overview
RDBMS Components:
  • Storage Engine - File-based storage with indexing
  • SQL Parser - Parses SQL-like queries
  • Query Executor - Executes parsed queries
  • Type System - Data type validation
  • REPL - Interactive command-line interface
Web App Components:
  • Flask Framework - Web server and routing
  • Templates - HTML with Bootstrap
  • Database Module - Integration with my MyRDBMS
  • CRUD Operations - Create, Read, Update, Delete
  • API Endpoints - JSON API for queries
