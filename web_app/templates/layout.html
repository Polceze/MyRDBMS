About MyRDBMS

This web application is powered by a custom-built relational database management system implemented in Python.

Features demonstrated:
  • SQL-like query language
  • CREATE TABLE with data types
  • CRUD operations (INSERT, SELECT, UPDATE, DELETE)
  • Primary and unique keys
  • Basic indexing
  • INNER JOIN operations
  • Interactive REPL (command-line)
  • Web interface (this app)

The entire database engine is built from scratch, including:

  • Storage engine with file persistence
  • SQL parser
  • Query executor
  • Index manager
